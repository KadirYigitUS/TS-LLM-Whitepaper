# Kaynak Zinciri

## İstatistikten Ajansa
- **N-gramlar** ve **vektör temsilleri**, anlamın matematiğe çevrildiği ilk katman.
- Transformer devrimi, uzun cümlelerdeki bağımlılıkları eşzamanlı takip etmeyi sağladı.

## Dokuz Makalenin Kısa Özeti
1. **Attention Is All You Need** — Özyinelemeyi bırak, dikkati kullan.
2. **BERT** — Maskelenmiş dil modeli ile çift yönlü bağlam.
3. **GPT-3** — Ölçek → bağlam içi öğrenme.
4. **Scaling Laws** — Kaynak planlaması için güç yasaları.
5. **LLaMA** — Akademi için açık ve verimli modeller.
6. **Chain-of-Thought** — Düşün-aloud promptları.
7. **InstructGPT** — RLHF hizalaması.
8. **Flan** — Talimat çeşitliliği.
9. **Toolformer** — API kullanımı ve araçsallaşma.

## Üçüncül Kaynakların Katkısı
- **Reiss & Vermeer / Nord**: Skopos ve sadakat → prompt şablonları.
- **Halliday**: Field/Tenor/Mode → bağlam enjeksiyonu.
- **Kenny, Bowker, House**: MT okuryazarlığı, etik, kültürel silinme.

## Araştırma Dönüşümleri
- BLEU yerine Skopos uyum puanı.
- Süreç verisinin (prompt, RAG paketleri, sıcaklık ayarları) arşivlenmesi.
- İnsan + AI “sibernetik” iş akışlarının izlenmesi.
