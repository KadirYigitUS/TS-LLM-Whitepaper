# TS-LLM Bilgi Yığını — Genel Bakış

Bu bölüm, Obsidian taslağının Türkçe özetini sunar ve çeviribilimci ekiplerin hızlıca doğru nota ulaşmasını sağlar.

## Hedef ve Kapsam
- **Amaç**: LLM mühendisliğinin tarihsel evrimini çeviribilim kuramlarıyla eşleştirmek.
- **Okuyucu**: Nitel TS birikimine sahip, ancak NLP kavramlarına yeni araştırmacılar.

## Giriş
- Üretken yapay zekâ çeviri iş akışlarına girdi; kara kutu algısı sürüyor.
- “Amaç temelli” açıklamalar, istatistiksel modelleri anlaşılır kılar.

## Dokuz Eşik Makale
1. **Transformer (Vaswani)** — Özyineliği bırakıp dikkat mekanizması kullanmak.
2. **BERT** — Maskelenmiş dil modeli ile çift yönlü bağlam.
3. **GPT-3** — Ölçek sayesinde bağlam içi öğrenme.
4. **Scaling Laws** — Performansın güç yasalarıyla öngörülebilmesi.
5. **LLaMA** — Açık ve verimli temel modeller.
6. **Chain-of-Thought** — Düşün-aloud protokollerinin prompt hâli.
7. **InstructGPT** — RLHF ile hizalama.
8. **Flan** — Talimat çeşitliliğiyle genelleme.
9. **Toolformer** — API kullanabilen araçsallaştırılmış modeller.

## Kuramsal Köprüler
- Skopos kuramı = Sistem promptu.
- Halliday bağlamı = RAG paketlerine dönüştürülen Field/Tenor/Mode.
- Eleştirel ampirizm = Prompta duyarlılık ölçümleri ve süreç denetimi.
